Adatkezelés XML-ben

A tantárgy bemutatása

Ez a kurzus az XML-alapú adatkezelési technikák elméletébe és gyakorlatába vezet be. A hallgatók megismerik az XML nyelv szabványait, a DTD és XMLSchema tervezés alapjait, és betekintést kapnak a DOM, SAX, és JAXB API-k működésébe. Emellett a kurzus során szó lesz az XSLT, XQuery és XML Security szabványok alkalmazásáról is.

A kurzus tematikája

1. hét – XML szabvány alapjai, Infoset és XDM modellek

2. hét – DTD szabvány, séma tervezés

3. hét – XMLSchema használata és elemei

4. hét – XMLSchema haladó technikák

5. hét – SAX API és Java alkalmazás készítése

6. hét – DOM API és Java alkalmazás készítése

7. hét – JAXB API és alkalmazások fejlesztése

8. hét – XSLT és XPath feldolgozás

9. hét – Haladó XSLT technikák

10. hét – XQuery alapok

11. hét – XQuery haladó funkciók

12. hét – XML Security és SVG alkalmazások

13. hét – XML alapú adatbázis kezelése, eXistDB

14. hét – Összefoglalás
