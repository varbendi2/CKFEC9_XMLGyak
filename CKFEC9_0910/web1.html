Web technológiák 1.

A tantárgy bemutatása

Ez a kurzus a webes technológiák alapjaiba nyújt bevezetést. A tantárgy keretében megismerjük a hálózati alapfogalmakat, a HTML, CSS és JavaScript használatát, valamint a webes alkalmazások fejlesztési módszereit. A hallgatók gyakorlati feladatokon keresztül sajátítják el a front-end technológiák alapjait, és képesek lesznek modern weboldalakat létrehozni.

A kurzus tematikája

1. hét – Hálózati alapfogalmak, protokollok

2. hét – Webes alkalmazások felépítése, fejlesztési módszerek

3. hét – HTML alapok

4. hét – HTML űrlapok, böngésző függőség

5. hét – CSS alapok

6. hét – CSS technikák

7. hét – Komplex feladat HTML és CSS segítségével

8. hét – JavaScript alapok

9. hét – JavaScript technikák

10. hét – AJAX és jQuery

11. hét – Web Design

12. hét – Komplex projekt HTML, CSS és JavaScript segítségével

13. hét – Összefoglalás és konzultáció
